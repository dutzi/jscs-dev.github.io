    disallowPaddingNewLinesBeforeLineComments

    Disallows newline before line comments

    Type: Boolean

    Value: true

    Example

    "disallowPaddingNewLinesBeforeLineComments": true
    Valid
    var a = 2;
// comment
return a;
    Invalid
    var a = 2;

//comment
return a;
