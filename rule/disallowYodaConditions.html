    disallowYodaConditions

    Requires the variable to be the left hand operator when doing a boolean comparison

    Type: Boolean

    Value: true

    Example

    "disallowYodaConditions": true
    Valid
    if (a == 1) {
    return
}
    Invalid
    if (1 == a) {
    return
}
