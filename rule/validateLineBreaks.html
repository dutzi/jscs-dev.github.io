    validateLineBreaks

    Option to check line break characters

    Type: String

    Values: "CR", "LF", "CRLF"

    Example

    "validateLineBreaks": "LF"
    Valid
    var x = 1;<LF>
x++;
    Invalid
    var x = 1;<CRLF>
x++;
