    disallowEmptyBlocks

    Disallows empty blocks (except for catch blocks).

    Type: Boolean

    Value: true

    JSHint: noempty

    Example

    "disallowEmptyBlocks": true
    Valid
    if ( a == b ) { c = d; }
try { a = b; } catch( e ){}
    Invalid
    if ( a == b ) { } else { c = d; }
