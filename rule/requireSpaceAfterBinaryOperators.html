    requireSpaceAfterBinaryOperators

    Disallows sticking binary operators to the right.

    Types: Array or Boolean

    Values: Array of quoted operators or true to require space after all possible binary operators

    Example

    "requireSpaceAfterBinaryOperators": [
    "=",
    ",",
    "+",
    "-",
    "/",
    "*",
    "==",
    "===",
    "!=",
    "!=="
    // etc
]
    Valid
    x + y;
    Invalid
    x +y;
