    disallowPaddingNewLinesInObjects

    Disallows newline inside curly braces of all objects.

    Type: Boolean

    Value: true

    Example

    "disallowPaddingNewLinesInObjects": true
    Valid
    var x = { a: 1 };
foo({a: {b: 1}});
    Invalid
    var x = {
    a: 1
};
foo({
    a: {
        b: 1
    }
});
