    requireSemicolons

    Requires semicolon after:

    • var declaration
    • expression statement
    • return
    • throw
    • break
    • continue
    • do-while

    Type: Boolean

    Value: true

    Example

    "requireSemicolons": true
    Valid
    var a = 1;
    Invalid
    var a = 1
