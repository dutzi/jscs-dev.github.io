    disallowIdentifierNames

    Disallows a specified set of identifier names.

    Type: Array

    Values: Array of strings, which should be disallowed as identifier names

    Example

    "disallowIdentifierNames": ['temp', 'foo']
    Valid
    var good = 1;
object['fine'] = 2;
object.fine = 3;
    Invalid
    var temp = 1;
object['foo'] = 2;
object.foo = 3;
