    disallowSpaceBeforeObjectValues

    Disallows space after object keys.

    Type: Boolean

    Value: true

    Example

    "disallowSpaceBeforeObjectValues": true
    Valid
    var x = {a:1};
    Invalid
    var x = {a: 1};
